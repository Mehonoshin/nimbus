Additional helpers for [Phoenix](https://github.com/phoenixframework/phoenix)

## Installation
1. Add `nimbus` to your list of dependencies in `mix.exs`:

  ```elixir
  def deps do
    [{:nimbus, "~> 0.1.0"}]
  end
  ```

2. Ensure `nimbus` is started before your application:

  ```elixir
  def application do
    [applications: [:nimbus]]
  end
  ```
